# Chocobot
Project management assistant deployes as a cloud-native web-app, integrates a Telegram Chatbot to enable mobile access to the platform.
Features: 
  -Agile Tracking: Full CRUD functionalities to transform raw requirements into actionable tasks with status tracking (To Do, In Process, Done).
  -Performance Analytics: A dedicated statistics module for visual analysis of project velocity and individual developer performance.
  -A specialized Telegram Chatbot integrated via API for real-time task updates and mobile interaction.
  -Automated multi-environment provisioning on Oracle Cloud Infrastructure (OCI) using Terraform (IaC), reducing infrastructure setup time and ensuring network consistency via custom VCN and subnet  configurations.
  -Integrated a CI/CD pipeline to enable automatic updates when any changes on the repository are detected. 
Framework: SpringBoot
